# Personal Finance Dashboard Agent

This is a simple personal finance dashboard built using Python and Streamlit.

## Features

- Upload bank statement CSV
- Automatically categorise transactions
- Calculate income, expenses, and savings
- Visualise spending patterns
- Monthly financial trends
- Simple financial insights

## Tech Stack

- Python
- Pandas
- Streamlit
- Plotly

## Goal

To help individuals understand their finances easily without needing a finance background.

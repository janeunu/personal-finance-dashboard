# Money Health Agent — Version 2

A clean personal finance dashboard for everyday users who want to understand their bank statement without finance knowledge.

## Main goal

Help users answer:

1. Am I okay financially?
2. How much money came in?
3. How much money went out?
4. Where did my money go?
5. What should I check next?

## Features

- Upload CSV bank statement
- Auto-categorise transactions
- Money Health Score
- Income, spending, savings, and daily spend cards
- Spending breakdown
- Income vs spending trend
- Top spending categories
- Plain-English insights
- Transaction explorer

## Run

```bash
pip install -r requirements.txt
